# Patient BMI Health Metrics System

This project demonstrates a simple Python program that calculates a patient's Body Mass Index (BMI), classifies the BMI category, assesses health risks, and generates a formatted health report.

### **2. BMI Calculation**
The `calculate_bmi()` method computes BMI using:
```
BMI = weight_kg / (height_m ** 2)
```
Includes basic validation to prevent division errors.

### **3. BMI Categorization**
The `get_bmi_category()` method returns categories including:
- Underweight
- Normal weight
- Overweight
- Obese Class I, II, III

### **4. Health Risk Assessment**
The `assess_risk()` method links each BMI category to a health risk explanation.

### **5. Report Generator**
The `display_report()` method prints a structured health report containing:
- Patient details
- BMI value
- BMI category
- Health risk assessment
